# Fox-Finder
A User-interface plugin for kismet that will help you find things.

Requires the most current version of kismet located over at:
https://github.com/kismetwireless/kismet.git

## Installation
clone this repo.

```cd foxfinder```

```sudo make install```

Happy hunting.
